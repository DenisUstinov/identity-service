# Идентификационный сервис

Микросервис аутентификации и управления пользователями.

## Статус проекта

Данный проект разрабатывался в учебных целях.

Разработка остановлена до достижения MVP и в дальнейшем не планируется к продолжению.
Репозиторий оставлен в текущем состоянии в качестве примера и архива учебной работы.
